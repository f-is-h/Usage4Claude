Claude AI 使用量を
リアルタイム監視

すべての Claude プラットフォームに対応したエレガントな macOS メニューバーアプリ

  • • 二重制限表示 - 5時間 + 7日間のクォータを一目で確認
  • • クロスプラットフォーム - Web/CLI/Desktop/Mobile の使用量を統一監視
  • • プライバシー優先 - ローカルストレージ、オープンソース、Keychain 暗号化
最新版をダウンロード ⭐ Star on GitHub

主な機能

Claude AI ユーザーのために設計されたプロフェッショナル監視ツール

📊

リアルタイム監視

メニューバーで使用量クォータをリアルタイム表示、5時間・7日間の二重制限対応、スマートカラーアラート（緑→橙→赤 / シアン→紫→ディープパープル）

🌐

クロスプラットフォーム対応

すべての Claude プラットフォームを統一監視：Web版、Claude Code、デスクトップアプリ、モバイルアプリ - 一箇所ですべての使用量を確認

🔄

スマート更新

4段階の適応型更新頻度、アクティブ時は1分間隔で高速更新、アイドル時は徐々に10分間隔まで減少

🌍

多言語対応

英語、日本語、簡体字中国語、繁体字中国語をネイティブサポート、インターフェース言語はシステムに自動追従または手動切替可能

🎨

パーソナライゼーション

3つの表示モード（アイコンのみ/パーセンテージのみ/アイコン+パーセンテージ）、3つのテーマ（デフォルト/モノクロ/カラー）、さまざまな好みに対応

🔒

セキュリティとプライバシー

すべてのデータはローカルにのみ保存、Session Key は Keychain で暗号化、100% オープンソースコード、自由に監査可能
