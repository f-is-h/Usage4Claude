Privacy Policy

Core Principle

We do not collect any user data.

Usage4Claude is a 100% local application. All data is stored only on your Mac.

Data Storage

Local Storage

  • Usage quota data: Cached locally after fetching from Claude API
  • App settings: Stored in local user preferences
  • No data uploaded to any external servers

Keychain Encryption

  • Organization ID and Session Key: Encrypted in macOS Keychain
  • Encryption: AES-256 + Hardware protection (T2/Secure Enclave)
  • Access Control: Only this app can access, other apps cannot read

Network Requests

Claude API Calls

  • Purpose: Fetch usage quota data
  • Frequency: Based on settings (1-10 min intervals)
  • Content: Only authentication info, no other data

No Tracking, No Analytics

  • ❌ No Google Analytics
  • ❌ No other analytics tools
  • ❌ No ad tracking
  • ❌ No third-party SDKs
  • ✅ Completely zero tracking

Open Source Transparency

  • Source Code: GitHub Repository
  • License: MIT License
  • Audit: Anyone can review and audit the code
  • No Obfuscation: Fully transparent, no hidden features

Data Security

  • ✅ App Sandbox Enabled - Limits system access
  • ✅ Keychain Encryption - Hardware-level protection for sensitive info
  • ✅ Minimal Permissions - Only requests necessary permissions (Network + Keychain)
  • ✅ No File Access - Does not access your files, contacts, etc.

Your Rights

  • Delete app and data anytime
  • View/delete credentials via "Keychain Access" app
  • No data residue concerns - all data stored locally only

Policy Changes

If this privacy policy changes, updates will be posted here and users will be notified in the app.
