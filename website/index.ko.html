Claude AI 사용량을
실시간 모니터링

모든 Claude 플랫폼을 지원하는 우아한 macOS 메뉴 바 앱

  • • 다중 제한 표시 - 최대 5가지 사용 제한 동시 표시
  • • 크로스 플랫폼 - Web/CLI/Desktop/Mobile 사용량 통합 모니터링
  • • 프라이버시 우선 - 로컬 저장소, 오픈 소스, Keychain 암호화
최신 버전 다운로드 ⭐ Star on GitHub

주요 기능

Claude AI 사용자를 위한 전문 모니터링 도구

📊

실시간 모니터링

최대 5가지 사용 제한 모니터링: 5시간, 7일, 추가 사용량, 7일 Opus, 7일 Sonnet. 구별하기 쉬운 다양한 모양의 스마트 색상 코딩 진행 링.

🌐

크로스 플랫폼 지원

모든 Claude 플랫폼 통합 모니터링: 웹 버전, Claude Code, 데스크톱 앱, 모바일 앱 - 한 곳에서 모든 사용량 확인

🔄

스마트 업데이트

4단계 적응형 업데이트 빈도, 활성 시 1분 간격으로 빠른 업데이트, 유휴 시 점진적으로 10분 간격까지 감소

🌍

다국어 지원

5개 언어 네이티브 지원: 영어, 일본어, 한국어, 간체 중국어, 번체 중국어. 인터페이스 언어는 시스템에 자동으로 따르거나 수동으로 전환 가능

🎨

개인화

스마트 모드로 사용 가능한 제한을 자동 감지하거나 사용자 정의로 표시할 제한을 선택. 3가지 테마(색상 반투명/색상 배경/단색)로 개인화 가능.

🔒

보안 및 프라이버시

모든 데이터는 로컬에만 저장, Session Key는 Keychain에서 암호화, 100% 오픈 소스 코드, 자유롭게 감사 가능
